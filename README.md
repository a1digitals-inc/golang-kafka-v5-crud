## Installation

### Prerequisites
- Golang(>=1.11) - Download and Install [Golang](https://golang.org/)
```
  $ git clone https://github.com/fitraaditama7/golang-kafka-v5-crud
  $ cd golang-kafka-v5-crud
  $ go mod download
  $ make consumer
  $ make producer
```
